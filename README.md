### Project: **Tasky Clone**
- **Description:** Built a feature-rich **To-Do List Application** using Flutter and Dart, designed to provide users with a seamless and efficient task management experience.
- **Key Features:**
  - **Task Management:** Add, edit, delete, and view tasks with ease.
  - **Filtering Options:** Apply filters to organize and prioritize tasks effectively.
  - **User Profile:** Personalize the experience with a dedicated user profile section.
  - **State Management:** Utilized the **BLoC pattern** for efficient and scalable state management.
  - **Modern UI:** Designed a clean and responsive user interface inspired by **Figma** prototypes.
  - **Figma Design URL**: https://www.figma.com/design/87DbfZRjJh3M5yYG7ntxe1/ToDoApp?node-id=0-1&p=f&t=kkAxKKBgsY3mc1xg-0
  - **Backend Integration:** Connected to RESTful APIs for backend communication.
- **Tools & Technologies:** Flutter, Dart.
